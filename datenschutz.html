2. Zweck der Datenverarbeitung

Diese Website dient der Vermittlung von Anfragen im Bereich Photovoltaik- und Solaranlagen. Personenbezogene Daten werden erhoben, um Interessenten an geeignete regionale Fachbetriebe weiterzuleiten.

3. Erhobene Daten

  • Vor- und Nachname
  • E-Mail-Adresse
  • Telefonnummer
  • Postleitzahl
  • Angaben zum Projekt (z. B. Gebäudetyp, Dachfläche, Zeitpunkt)

4. Rechtsgrundlage

Die Verarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. b DSGVO (Vertrag / vorvertragliche Maßnahmen) sowie Art. 6 Abs. 1 lit. a DSGVO (Einwilligung).

5. Weitergabe an Dritte

Die übermittelten Daten werden an ausgewählte regionale Fachbetriebe weitergegeben, damit diese Kontakt aufnehmen und ein Angebot erstellen können.

6. Hosting

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf werden technisch notwendige Server-Log-Daten verarbeitet.

7. Formularanbieter

Zur Erfassung der Anfragen wird der Dienst „Tally“ verwendet. Die Daten werden über die Server des Anbieters verarbeitet.

8. Speicherdauer

Personenbezogene Daten werden nur so lange gespeichert, wie es zur Vermittlung der Anfrage oder aufgrund gesetzlicher Pflichten erforderlich ist.
